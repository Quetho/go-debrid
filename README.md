go-debrid
===========

Tool written in Go to interact with Alldebrid API

How to use
----------

* Debrid a link supported by Alldebrid
```
	-d	<link>
```
* Add a torrent
```
	-t	<torrent file>
```
* Check torrent list
```
	-l
```
* Remove a torrent (tip: use "```-l```" and check column ID to get the entry to remove)
```
	-r	<torrent id>
```
